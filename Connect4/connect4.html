Connect-4

Welcome to Connect-4

    RULES:
  • Select any column to play your move.
  • Move can only be played in the last empty row.
    • To WIN:
  • Connect four values in any direction:
    1. Horizontal
    2. Vertical
    3. Primary Diagonal
    4. Secondary Diagonal
    Difficulty:
  • Smaller boards are considered as easy.
  • 6x7 is considered as normal.
  • Bigger boards make the game lengthy and diffcult.

Select the board size to start your game!
